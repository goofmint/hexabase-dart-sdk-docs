`hexbase-sdk` is essentially npm package, you can install it by `npm`

```bash
npm i hexabase-sdk
```

## Requirements

* [Nodejs](https://nodejs.org/en/) >= 10

* user credentials ***must obtain from hexabase***

```json
{
    "email": "j.soliva@hexabase.com",
    "password": "123456"
}
```
